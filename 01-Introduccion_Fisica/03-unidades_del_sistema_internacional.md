# 3. UNIDADES DEL SISTEMA INTERNACIONAL (SI)

---

## 3.1 ¿Qué es el Sistema Internacional (SI)?

El Sistema Internacional de Unidades (SI) es el sistema de medición utilizado a nivel mundial en la ciencia y la tecnología.

Su función es establecer unidades estándar para medir magnitudes físicas, permitiendo que los resultados sean comparables en cualquier parte del mundo.

---

## 3.2 Unidades fundamentales del SI

El Sistema Internacional define siete unidades fundamentales, que sirven como base para todas las demás.

| Magnitud               | Unidad        | Símbolo |
|-----------------------|--------------|--------|
| Longitud              | metro        | m      |
| Masa                  | kilogramo    | kg     |
| Tiempo                | segundo      | s      |
| Temperatura           | kelvin       | K      |
| Corriente eléctrica   | ampere       | A      |
| Cantidad de sustancia | mol          | mol    |
| Intensidad luminosa   | candela      | cd     |

---

## 3.3 Características de las unidades fundamentales

- Son independientes entre sí.
- No se definen a partir de otras unidades.
- Permiten construir todas las unidades derivadas.

---

## 3.4 Unidades derivadas del SI

Las unidades derivadas se obtienen a partir de combinaciones de las unidades fundamentales.

| Magnitud   | Unidad       | Símbolo| Expresión en unidades básicas|
|------------|--------------|--------|------------------------------|
| Velocidad  | metro/segundo| m/s    | m·s⁻¹                        |
| Aceleración| metro/seg²   | m/s²   | m·s⁻²                        |
| Fuerza     | newton       | N      | kg·m·s⁻²                     |
| Energía    | joule        | J      | kg·m²·s⁻²                    |
| Potencia   | watt         | W      | kg·m²·s⁻³                    |
| Presión    | pascal       | Pa     | kg·m⁻¹·s⁻²                   |
| Densidad   | kg/m³        | kg/m³  | kg·m⁻³                       |

---

## 3.5 Importancia del Sistema Internacional

El uso del SI es fundamental porque:

- Unifica las mediciones a nivel mundial.
- Evita confusiones entre diferentes sistemas de unidades.
- Facilita el trabajo científico y tecnológico.
- Permite expresar resultados de manera clara y precisa.

---

## 3.6 Ejemplo de uso

Si una fuerza es de 10 N, esto significa:

10 N = 10 kg·m·s⁻²  

Es decir, se puede expresar en función de las unidades fundamentales.

---

## 3.7 Conclusión

El Sistema Internacional de Unidades proporciona las bases para medir y expresar magnitudes físicas de forma universal, permitiendo desarrollar y comunicar el conocimiento científico de manera precisa.
